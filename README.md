# How to run 
- open an intergrated terminal with the server.js
- run mysql -u root -p 
- login to your mysql
- source the schema.sql
- source the seeds.sql
- once done exit with \q
- run npm i (to install the dependencies)
- run node server.js
- to navigate use up and down arrows to select options
